## Back-End Development

### 1. OpenAI API Integration

- **Set Up OpenAI API Access:**
  - Obtain an API key from OpenAI.
  - Store the API key securely in Replit's environment variables.

- **Create a Function for API Calls:**

  ```python
  import openai
  openai.api_key = os.getenv('OPENAI_API_KEY')

  def generate_response(prompt, conversation_history):
      response = openai.ChatCompletion.create(
          model="gpt-4",
          messages=conversation_history + [{"role": "user", "content": prompt}]
      )
      return response['choices'][0]['message']['content']
  ```

### 2. PDF Ingestion and Summarization

- **Create a Folder for Readings:**
  - Name the folder `Readings`.
  - Ensure it's only accessible by the professor or backend superuser.

- **Implement PDF Reading Functionality:**

  ```python
  import PyPDF2

  def extract_text_from_pdfs():
      readings_folder = 'Readings'
      text_content = ""
      for filename in os.listdir(readings_folder):
          if filename.endswith('.pdf'):
              with open(os.path.join(readings_folder, filename), 'rb') as f:
                  reader = PyPDF2.PdfReader(f)
                  for page in reader.pages:
                      text_content += page.extract_text()
      return text_content
  ```

- **Summarize Important Concepts:**

  - Use NLP techniques or OpenAI's API to summarize the text.

  ```python
  def summarize_text(text):
      summary = openai.ChatCompletion.create(
          model="gpt-4",
          messages=[{"role": "system", "content": "Summarize the key concepts of the following text."},
                    {"role": "user", "content": text}]
      )
      return summary['choices'][0]['message']['content']
  ```

### 3. Question Generation

- **Generate Starting Questions from Summaries:**

  ```python
  def generate_starting_questions(summary):
      prompt = f"Based on the following summary, create thought-provoking Socratic questions:\n\n{summary}"
      response = openai.ChatCompletion.create(
          model="gpt-4",
          messages=[{"role": "user", "content": prompt}]
      )
      questions = response['choices'][0]['message']['content']
      return questions.split('\n')
  ```

### 4. Socratic Dialogue Logic

- **Implement Dialogue Flow:**

  - Maintain conversation history.
  - Ensure the bot follows the custom instructions for Socratic dialogue.

  ```python
  def socratic_dialogue(user_input, conversation_history):
      # Append user input to conversation history
      conversation_history.append({"role": "user", "content": user_input})

      # Generate bot response with custom instructions
      response = openai.ChatCompletion.create(
          model="gpt-4",
          messages=conversation_history
      )
      bot_reply = response['choices'][0]['message']['content']
      conversation_history.append({"role": "assistant", "content": bot_reply})
      return bot_reply, conversation_history
  ```

- **Incorporate Custom Instructions:**

  - Embed the custom instructions in the system prompt at the beginning of the conversation.

  ```python
  custom_instructions = """
  [Your custom instructions here]
  """

  conversation_history = [
      {"role": "system", "content": custom_instructions}
  ]
  ```

### 5. User Authentication

- **Implement User Registration and Login:**

  - Use `streamlit-authenticator` for authentication.

  ```python
  import streamlit as st
  import streamlit_authenticator as stauth

  # Define users
  users = {'usernames': {'student1': {'name': 'Student One', 'password': 'hashed_password1'},
                         'student2': {'name': 'Student Two', 'password': 'hashed_password2'}}}

  authenticator = stauth.Authenticate(users, 'app_home', 'auth', cookie_expiry_days=30)

  name, authentication_status, username = authenticator.login('Login', 'main')

  if authentication_status:
      st.write(f'Welcome {name}')
  elif authentication_status == False:
      st.error('Username/password is incorrect')
  elif authentication_status == None:
      st.warning('Please enter your username and password')
  ```

  - **Note:** Replace `'hashed_password'` with actual hashed passwords.

### 6. Transcript Generation

- **Store Conversation History:**
  - Maintain a log of the conversation.

- **Implement Download Feature:**

  ```python
  def download_transcript(conversation_history):
      transcript = ""
      for msg in conversation_history:
          role = 'You' if msg['role'] == 'user' else 'QuizBot'
          transcript += f"{role}: {msg['content']}\n\n"
      st.download_button('Download Transcript', transcript, file_name='transcript.txt')
  ```
